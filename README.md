# engimon-java
How to run:

``` gradlew run``` atau ```./gradlew run```

NOTE: Jika tidak muncul gambar saat dirun, mark folder resources (```src/main/resources```) sebagai Resources Root. Caranya buka project dengan IntelliJ Idea, klik kanan di folder resources, lalu pilih mark directory as resources root.
